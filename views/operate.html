Main Networks Programming Operating Systems


    Operating systems are software programs that allow us to utilize devices to suit our needs. Below is a list of some operating systems that are or used to be popular.

  1. MS-DOS: An old operating system released in 1981 that stopped being produced and used in 2000.
  2. Microsoft Windows: Released by Microsoft in 1985, then became the most popular operating system. Products include windows 95, xp, 7, and 10.
  3. Mac OS: Released by apple in 2001, it's the primary os of apple computers. Products include mac os x, os x, and macOS.
  4. Android: Released by Google in 2007, it quickly became the most popular operating system for mobile phones. Previous software versions were named after candies, such as lollipop and kitkat.
  5. iOS: Released by apple in 2007 for iphones. It is the second most popular operating system for mobile phones. The latest version is ios13. Source
